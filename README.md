# 🗓️ Sistema Agenda

Sistema de agendamento desenvolvido com **React + Vite + TypeScript**, utilizando uma arquitetura moderna para consumo de APIs, gerenciamento de estado e formulários.

---

## 🚀 Tecnologias

Este projeto foi construído com as seguintes tecnologias:

- ⚛️ React 19
- ⚡ Vite
- 🟦 TypeScript
- 🎨 Bootstrap 5
- 🌐 Axios
- 🧠 Zustand (state management)
- 📝 React Hook Form
- 🔀 React Router
- 🧪 JSON Server (mock API)
- 🧹 ESLint
